•Start with file main.py

If the model doenst exist, then will create a new model automatically just running file main.py. The model need 5 class (rock, paper, scissors, ok, negative). If the program cant find the classes, then will open file saving_imgs.py for take 500 photos for each class. After that, the model will create and finally the file game.py will run to play.

Just run main.py :)
